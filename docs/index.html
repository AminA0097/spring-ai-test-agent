🚀 Spring AI Test Agent

AI-powered GitHub agent that generates, runs, and fixes Spring Boot tests automatically.

⚡ What it does

  • Analyzes Spring Boot projects
  • Generates JUnit5 + Mockito tests
  • Runs tests in isolated Docker environment
  • Fixes failing tests automatically
  • Creates GitHub Pull Requests

🔁 How it works

GitHub Push
   ↓
Analyze Repo
   ↓
Generate Tests
   ↓
Run Tests
   ↓
Fix Errors (AI Loop)
   ↓
Create PR

🧠 Built for Spring Boot teams

Turn your repository into a self-testing system.
